# Вгадай число (Guess the Number)

Мінімалістична та преміальна веб-гра «Вгадай число» від 1 до 1000. Комп'ютер вгадує ваше число максимум за 10 запитань, використовуючи логіку бінарного пошуку.

## Особливості
- **Преміальний дизайн**: Естетика Glassmorphism з Outfit шрифтом.
- **Адаптивність**: Коректне відображення на мобільних пристроях та десктопах.
- **Українська локалізація**: Повністю україномовний інтерфейс та брендинг.

## Як запустити
Просто відкрийте `index.html` у вашому браузері.

## Технології
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
