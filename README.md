# Вгадай число (Guess the Number) — C# Console Version

Це чиста консольна версія гри «Вгадай число» від 1 до 1000, написана на мові C#. Програма виступає у ролі того, хто вгадує, і гарантовано знаходить загадане вами число за 10 або менше запитань.

## Особливості
- **Чистий C#**: Реалізовано як .NET 8 консольний додаток.
- **Алгоритм**: Використовує ефективний бінарний пошук.
- **Україномовний інтерфейс**: Повна підтримка UTF-8 для коректного відображення в терміналі.

## Як запустити
Для запуску проекту вам потрібно мати встановлений **.NET SDK** (версії 6.0, 7.0 або 8.0).

1. Перейдіть до папки проекту:
   ```bash
   cd 1-1000
   ```
2. Запустіть програму:
   ```bash
   dotnet run
   ```

## Правила гри
1. Загадайте число від 1 до 1000.
2. Відповідайте на питання програми:
   - Вводьте **так**, **т** або **+**, якщо число БІЛЬШЕ за вказане.
   - Вводьте **ні**, **н** або **-**, якщо число МЕНШЕ або ДОРІВНЮЄ вказаному.
3. Отримайте свій приз після вгадування!
